## Introduction
A toDo CRUD application.

## Tech Stack
- Laravel

## Environment
```
php artisan --version
Laravel Framework 10.3.3
```

## Configuration
| file | field | value |
| --- | --- | --- |
| .env | DB_DATABASE | database name |
| | DB_USERNAME | mysql user name |
| | DB_PASSWORD | mysql user password | 

## Setup
```
php artisan migrate
php artisan serve
```
